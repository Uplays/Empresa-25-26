# Empresa 25-26 - Proyecto Scout

 **Sitio web oficial del proyecto "Empresa 25-26"** - Un proyecto de Escultas para un intercambio scout en Francia.

##  Características

- **Diseño Premium** con efectos glassmorphism y animaciones suaves
- **Tienda 3D Interactiva** con rotación de productos
- **Totalmente Responsive** - Funciona en móviles, tablets y desktop
- **Optimizado** para rendimiento y velocidad
- **Seguro** con headers de seguridad implementados
- **Accesible** con navegación por teclado y aria-labels

## Ver en Vivo

Visita la web: [Empresa-25-26](https://uplays.github.io/Empresa-25-26/)
